# Roadmap

This repository is community-maintained and is not an officially supported Tenable project.

## Current

- `v0.1.0` released with core MCP tooling and deployment guidance.

## Next Milestone: v0.1.1

- Focus on bug fixes, edge-case handling, and documentation clarity.
- Improve issue triage using `bug` + `priority:*` labels.
- Cut patch release after regression fixes are validated in CI.

## Next Milestone: v0.2.0

- Add non-breaking feature enhancements based on issue backlog.
- Expand automated tests for request/response shaping behavior.
- Improve operator docs for secure remote deployments.

## Ongoing

- Review new issues weekly.
- Prioritize high-impact bugs and security-sensitive reports.
- Keep changelog and release notes current for each tag.
