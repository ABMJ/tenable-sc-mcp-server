# Contributing

Thanks for using and improving this MCP server.

This repository is community-maintained and is not an officially supported Tenable project.

## Reporting Bugs

Use the bug report template:

- <https://github.com/ABMJ/tenable-sc-mcp-server/issues/new?template=bug_report.yml>

Please include:

- MCP server version (release tag, for example `v0.1.0`)
- Transport mode (`stdio` or `streamable-http`)
- Tenable.sc version if known
- Steps to reproduce
- Expected behavior
- Actual behavior
- Relevant logs or API error details (remove sensitive data)

## Suggesting Enhancements

Use the feature request template:

- <https://github.com/ABMJ/tenable-sc-mcp-server/issues/new?template=feature_request.yml>

Describe the use case first, then the proposed behavior.

## Security Reports

Do not post credentials, access keys, secret keys, or internal hostnames in public issues.

If you discover a sensitive security issue, open an issue with minimal details and request a private contact channel.

## Development Notes

- Keep changes focused and small when possible.
- Update `README.md` and `CHANGELOG.md` when behavior changes.
- Prefer adding issue references in pull requests (for example `Closes #12`).
