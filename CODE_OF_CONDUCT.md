# Code of Conduct

This repository is community-maintained and is not an officially supported Tenable project.

## Our Standards

This project aims to be welcoming, respectful, and practical.

Examples of positive behavior:

- Being respectful in discussions and reviews
- Giving constructive technical feedback
- Assuming good intent while clarifying disagreements

Examples of unacceptable behavior:

- Harassment, insults, or personal attacks
- Publishing private/sensitive data without consent
- Disruptive behavior that derails collaboration

## Scope

This Code of Conduct applies to project spaces, including issues, pull requests, and discussions tied to this repository.

## Enforcement

If you experience or observe unacceptable behavior, open an issue with the prefix `conduct:` and include enough detail to review the concern.

Project maintainers may remove, edit, or reject comments/issues/PRs that violate this policy.
