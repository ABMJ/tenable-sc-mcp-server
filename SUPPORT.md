# Support

## Where To Get Help

- Bug: <https://github.com/ABMJ/tenable-sc-mcp-server/issues/new?template=bug_report.yml>
- Feature request: <https://github.com/ABMJ/tenable-sc-mcp-server/issues/new?template=feature_request.yml>
- Existing issues: <https://github.com/ABMJ/tenable-sc-mcp-server/issues>

## What To Include

- release version/tag (for example `v0.1.0`)
- transport mode (`stdio` or `streamable-http`)
- Tenable.sc version (if known)
- minimal reproduction steps
- redacted logs/error payload

## Response Expectations

This is a community-maintained project. Triage and fixes are best-effort and prioritized by severity and impact.
